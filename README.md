# CS-370
For Project Two, the maze environment and overall training structure were provided. My contribution was implementing the qtrain() function, which is where the agent actually learns how to navigate the maze through trial, error, and gradual improvement. This project reflects how computer scientists approach problems generally: understand what the system needs to do, break it into manageable pieces, and refine based on what works. Ethically, any intelligent system should be honest and reliable for the people using it, and built with the understanding that real deployment carries consequences a classroom setting does not.
